### Задание 1
На лекции рассматривались режимы репликации master-slave, master-master, опишите их различия.
Ответить в свободной форме.
Репликация Master-Slave — это схема организации базы данных, при которой есть один главный сервер (Master) и один или несколько подчинённых серверов (Slave). Master принимает все операции записи (добавление, изменение, удаление данных), а Slaves только читают данные и получают обновления от Master.
Репликация Master-Master — это метод организации работы с базами данных, при котором несколько серверов выступают в роли главных (master), то есть каждый из них может одновременно принимать и обрабатывать запросы на запись и чтение данных. Данные между такими серверами автоматически синхронизируются, чтобы оставаться одинаковыми.
